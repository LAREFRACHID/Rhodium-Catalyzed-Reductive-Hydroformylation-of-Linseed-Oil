Multi-Output Regression Pipeline

This project predicts chemical outputs (ALD, ALC, CONJ, SAT) from experimental conditions using machine learning and explains the outputs using SHAP.
Models:
- Linear Regression
- SVR
- Random Forest
- XGBoost
- MLP

Features:
- Multi-output prediction
- Hyperparameter tuning
- Metrics (R², RMSE, MAE)
- Model comparison
- SHAP interpretability

Input:
- Excel file: exp_data.xlsx

Run:
python script.py

Output:
- Results saved in results_regression/
